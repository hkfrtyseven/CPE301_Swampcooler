# CPE301_Swampcooler
Implementation of a swamp cooler using Arduino

src/main.cpp

Arduino Pin Connections : 

PB6 - Green LED

PB5 - Red LED

PB4 - Yellow LED

PH6 - Blue LED

PF0 - Water Sensor S

PH4 - Temp Sensor S

PK7 - Standby button

PH3 - LCD RS

PE3 - LCD E

PG5 - LCD D4

PE5 - LCD D5

PH5 - LCD D6

PE4 - LCD D7
